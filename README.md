## Software used 
- NodeJS 
- ReactJS 
- Firebase platform

## How to run: 
1. Gather all files in local repo 
```
git clone https://github.com/albert-fung/Project-Bee.git
```

2. Go to source file and download all dependencies/node-modules 
```
npm install
```

3. To run local react app 
```
npm start
```
